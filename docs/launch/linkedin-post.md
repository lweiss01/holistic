**The context tax is real.**

If you use more than one AI coding assistant, you've felt it: every new session starts with re-explaining the project. Bugs come back because the next agent doesn't know what was already fixed. Progress gets lost when context windows end. Agents undo each other because there's no durable handoff.

**Holistic fixes that by making your repo the source of truth.**

![Holistic demo: before and after workflow comparison](https://github.com/lweiss01/holistic/raw/main/docs/demo.gif)

Your AI agents get shared memory inside the repo itself. When you switch from Claude to Codex to Gemini, the next agent can see what happened last time, what not to break, and what should happen next.

**One command. Every agent. Zero re-explaining.**

```bash
npm install -g holistic
holistic bootstrap
```

Then open your repo in any agent app and ask it to read `AGENTS.md` and `HOLISTIC.md`. That's it. The agent picks up where you left off without a long re-brief.

Built for people who use more than one AI assistant and are tired of paying the context tax.
