**The context tax is real.**

If you use more than one AI coding assistant, you've felt it.

Every new session starts with re-explaining the project. Bugs come back because the next agent doesn't know what was already fixed. Progress evaporates when context windows end. Agents undo each other because there's no durable handoff.

I built **Holistic** to fix this — by making your repo the source of truth for every agent you use.

**One command. Every agent. Zero re-explaining.**

```bash
npm install -g holistic
holistic bootstrap
```

Open your repo in Claude, Codex, Gemini, Cursor, or any other assistant. Ask it to read `AGENTS.md` and `HOLISTIC.md`. The agent picks up exactly where you left off — goal, status, what not to break — without a long re-brief.

Holistic works through repo files, so it's model-agnostic and device-portable by design. It also now includes `holistic repair` for anyone who hits path issues after moving a repo — a common rough edge, now a one-command fix.

We're in early beta and already seeing real-world use. If this solves a problem you have, I'd love to hear about it.
